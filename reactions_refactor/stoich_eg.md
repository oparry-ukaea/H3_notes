## Reactions

H ionisation, recombination; He ionisation, recombination:

|     |                        |
| --- | ---------------------- |
| R1  | $H + e \to H^+ + 2e$   |
| R2  | $H^+ + e \to H$        |
| R3  | $He + e \to He^+ + 2e$ |
| R4  | $He^+ + e \to He$      |
